Calorie Counter — Mark 1 (Flat build, NO SUBFOLDERS)
---------------------------------------------------
• Single-folder static site. Runs 100% in the browser.
• OCR via Tesseract.js CDN. Meal estimator with small local DB.

Files:
- index.html
- styles.css
- app.js

Deploy on Vercel (no CLI):
1) Create a new GitHub repo.
2) Upload these three files to the repo root.
3) In Vercel: New Project → Import the repo → Deploy.

Local test:
- Double-click index.html to open in your browser.
